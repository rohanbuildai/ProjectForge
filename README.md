# 🚀 ProjectForge

ProjectForge is a production-ready collaborative project management platform inspired by tools like Jira, Trello, and Notion.

It enables teams to manage workspaces, projects, tasks, and collaboration through a secure REST API built with Node.js, Express.js, and PostgreSQL.

---

## ✨ Features

- User Authentication (JWT + Refresh Tokens)
- Workspace Management
- Team Collaboration
- Project Management
- Task Management
- Role-Based Access Control (RBAC)
- Secure REST API
- Production-Ready Backend Architecture

---

## 🛠 Tech Stack

- Node.js
- Express.js
- PostgreSQL
- JWT
- bcrypt
- Cookie Parser
- dotenv
